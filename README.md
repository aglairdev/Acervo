# Acervo ꕤ

Buscador de metadados de preservação digital.

## Features

- Filtro em tempo real com debounce.
- Conversão dinâmica de unidades.
- Zero cookies, ads ou tracking.

## Config

Para o funcionamento do motor de busca, é necessário popular o diretório `data/` com arquivos JSON seguindo a estrutura abaixo:

```json
[
  {
    "name": "",
    "console": "",
    "size": "",
    "link": ""
  }
]
```
